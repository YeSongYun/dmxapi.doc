# 我们的AI探索之路

## 早期探索：从传统模型到AI变革

我们对AI技术的热情可以追溯到2018年，那时我们就开始利用百度EasyDL平台搭建各类机器学习模型，并将其成功融入到实际业务场景中。这段早期的实践经历为我们后来的AI发展奠定了坚实的技术基础，也让我们深刻理解了AI技术在商业应用中的巨大潜力。

## 创业思考：四大AI核心方向

2023年，我们首先思考创业方向，很快确定了四个方向：**客户端应用**、**API贸易**、**算力贸易**以及**数据处理**。这四个方向我们都涉及并实践摸索。

## 战略调整：聚焦核心竞争力

经过半年的实践探索过程，我们意识到**模型能力才是整个AI行业未来三年的核心竞争力**。正是这一关键洞察让我们重新审视了市场时机和自身定位。基于对行业发展节奏的深刻理解，我们果断调整了战略重心，最终聚焦到 API 领域。

## 创业探索：四个领域都探索、都实践

2018年9月，使用百度 EasyDL 训练小模型，融入业务流。

2022年12月，开始使用 ChatGPT。

2023年4月，开始摸索 RAG 应用，6周后意识到模型能力的时间点还没到，果断放弃 RAG 方向。

2023年5月，正式上线[Langchain.com.cn](http://www.Langchain.com.cn)社群是我们在AI领域的重要尝试。

2023年7月，尝试算力贸易业务，去上海AI大会发传单卖H100；聊过70亿买家、深圳仓库、矿机大佬，最终发现贸易型不适合我们。

2024年1月，开始API贸易，TO B 为主。

2024年5月，尝试数据处理业务，深度思考后明确了数据处理目标，并且和投资人热情高涨的聊了10个小时，但调研后发现在中国还没到需求时间点。

2024年7月，DMXAPI.com 国际站正式上线，很快获得正反馈。

2025年2月，DMXAPI.cn 国内站正式上线。

2025年5月，SSVIP.DMXAPI.com 生产级别站上线。

...


## 商业突破：DMXAPI聚合平台

2024年春节，我们正式启动了API贸易业务，这标志着我们商业化进程的重要里程碑。经过数月的精心打磨和优化，在2024年7月，我们的**DMXAPI中国大模型聚合平台**正式上线，为用户提供一站式的AI API服务解决方案。
2022年创办上海数员的时候，是希望20年后可以拥有100万个自动化程序服务社会，而 DMXAPI 将接过这个目标勇敢前进。
