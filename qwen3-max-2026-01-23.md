# qwen3-max-2026-01-23 工具调用API文档
**qwen3-max-2026-01-23**  通义千问3系列Max模型，此版本实现思考模式和非思考模式的有效融合，模型整体效果得到全方位的大幅度提升。在思考模式下，同时具备Web搜索、Web信息提取和代码解释器工具能力，使得模型在慢思考的同时，能够通过引入外部工具，以更高的准确性解决更有难度的问题。

## 接口地址 
```
https://www.dmxapi.cn/v1/chat/completions
```

## 模型名称
`qwen3-max-2026-01-23 `

## 简单对话调用示例
请参考 [OpenAI 流式输出 API 文档 ](https://doc.dmxapi.cn/openai-chat-stream.html)
## 工具调用示例
```python
import requests
import os

url = "https://www.dmxapi.cn/v1/chat/completions"
headers = {
    "Authorization": "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    "Content-Type": "application/json"
}
data = {
    "model": "qwen3-max-2026-01-23",
    "messages": [
        {
            "role": "system",
            "content": "You are a helpful assistant."
        },
        {
            "role": "user",
            "content": "上海明天天气如何"
        }
    ],

    """
    【参数】 enable_search
    该参数设置为True可启用联网搜索功能。启用后，模型将判断用户问题是否需要联网查询：
    若需要，则结合搜索结果回答；若不需要，则直接使用模型自身知识回答。
    """
    "enable_search": True 


}

response = requests.post(url, headers=headers, json=data)
print(response.json())


```
