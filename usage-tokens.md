# 豆包计费规则明细

## 豆包视频计费规则

视频生成模型采用异步推理，将输出视频转化为 token 计费，按 token 后付费价格。

### 计费公式

```
推理费用 = 按token付费单价 × 视频token用量
视频token用量 ≈ (宽 × 高 × 帧率 × 时长) / 1024
```

> **注意**：准确的 token 用量以调用 API 后返回信息中的 `usage` 字段为准。该计费为厂商计费，DMXAPI并未参与tokens计算，如果您不信任该厂商，请勿使用该厂商模型！
